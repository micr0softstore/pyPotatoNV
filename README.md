How to use (GUI):

python3 -m pip -r requirements.txt --break-system-packages (Will install the modules locally)

python3 app.py


How to use (Command line):

python3 main.py --chipset [for example hisi960] --unlock


Will get the unlock code


python3 main.py --chipset [for example hisi960] --wipefrp


Will erase frp lock.
