How to use (GUI):\n
python3 -m pip -r requirements.txt --break-system-packages (Will install the modules locally)\n
python3 app.py\n
\n
How to use (Command line):\n
python3 main.py --chipset [for example hisi960] --unlock\n
\n
Will get the unlock code\n
\n
python3 main.py --chipset [for example hisi960] --wipefrp\n
\n
Will erase frp lock.
